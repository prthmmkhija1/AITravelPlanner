## Design System

### Color Palette
```
Primary Blue:       #1e88e5  (Trust, Travel, Actions)
Dark Blue:          #1565c0  (Headers, Emphasis)
Success Green:      #4caf50  (Price drops, Confirmations)
Warning Yellow:     #f9c74f  (CTAs, Highlights)
Danger Red:         #ff4444  (Live indicators, Alerts)
Neutral Gray:       #666    (Muted text)
Light Background:   #f8fafc (Cards, Panels)
Border Gray:        #e2e8f0 (Dividers, Edges)
```

### Typography
- **Font Family:** System UI fonts (Segoe UI, Roboto, Helvetica)
- **Headings:** 700-800 weight, size 1.2rem - 2.5rem
- **Body:** 400-600 weight, size 0.9rem - 1rem
- **Small:** 0.8rem - 0.85rem (labels, captions)

### Spacing
- **Component Gap:** 16px (standard spacing)
- **Section Padding:** 24px - 64px
- **Border Radius:** 8px (buttons), 12px (cards), 16px (modals), 25px (rounded buttons)

## Interactions & Micro-interactions

### Button States
```
Default:        Blue background, white text
Hover:          Darker blue, lifted (2px), shadow glow
Active/Click:   Pressed appearance (no lift)
Disabled:       Opacity 65%, no cursor change
```

### Card Hover Effects
```
Default:        Flat, minimal shadow
Hover:          Lifted 4-8px, enhanced shadow, scale 1.02
```

### Input States
```
Default:        Light border, white background
Focus:          Blue border, blue shadow ring
Error:          Red border, light red background
```

### Loading States
```
- Spinning circle (CSS animation)
- "Loading..." text below spinner
- Modal content grayed out
- Centered in content area
```

## Responsive Breakpoints

### Mobile (< 480px)
- Single column layouts
- Full-width modals
- Stacked buttons
- Compact margins (12px)
- Text size adjusted with clamp()

### Tablet (480px - 768px)
- 2-column grids
- Modals with padding
- Flexbox wrapping
- Standard margins (16px)

### Desktop (> 768px)
- 3-4 column grids
- Full modals with max-width
- Side-by-side layouts
- Generous spacing (24px+)
- Hover states active
